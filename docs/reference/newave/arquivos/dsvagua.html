Outros Usos da Água (dsvagua.dat)¶

Visão geral do modelo¶

Os desvios para usos consuntivos da água no modelo NEWAVE, localizados no arquivo geralmente denominado dsvagua.dat, são armazenados na classe:

class DSVAgua(tabela: numpy.ndarray)[código fonte]¶

Armazena os dados de entrada do NEWAVE referentes aos desvios de água por usina.

Esta classe lida com informações de entrada fornecidas ao NEWAVE e que podem ser modificadas através do arquivo dsvagua.dat.

Parâmetros

  • tabela: np.ndarray

Leitura¶

A leitura do arquivo dsvagua.dat é feita através da classe LeituraDSVAgua.

class LeituraDSVAgua(diretorio: str)[código fonte]¶

Realiza a leitura do arquivo dsvagua.dat existente em um diretório de entradas do NEWAVE.

Esta classe contém o conjunto de utilidades para ler e interpretar os campos de um arquivo dsvagua.dat, construindo um objeto DSVAgua cujas informações são as mesmas do dsvagua.dat.

Este objeto existe para retirar do modelo de dados a complexidade de iterar pelas linhas do arquivo, recortar colunas, converter tipos de dados, dentre outras tarefas necessárias para a leitura.

Uma vez realizada a leitura do arquivo, as informações são guardadas internamente no atributo dsvagua.

Exemplos

>>> diretorio = "~/documentos/.../deck"
>>> leitor = LeituraDSVAgua(diretorio)
>>> leitor.le_arquivo()
# Ops, esqueci de pegar o objeto
>>> dsv = leitor.dsvagua
le_arquivo() → inewave.newave.modelos.dsvagua.DSVAgua[código fonte]¶

Faz a leitura do arquivo dsvagua.dat.

Escrita¶

A escrita do arquivo dsvagua.dat é feita através da classe EscritaDSVAgua.

class EscritaDSVAgua(diretorio: str)[código fonte]¶

Realiza a escrita do arquivo dsvagua.dat em um diretório de entradas do NEWAVE.

Esta classe contém o conjunto de utilidades para escrever os campos de um arquivo dsvagua.dat, a partir de um objeto DSVAgua.

Este objeto existe para retirar do modelo de dados a complexidade de armazenar as strings auxiliares do arquivo, desenhar tabelas, dentre outras tarefas associadas à escrita.

Se o diretório de escrita não existir, ele será criado.

Exemplos

>>> diretorio = "~/documentos/.../deck"
>>> # dsv é do tipo DSVAgua
>>> escritor = EscritaDSVAgua(diretorio)
>>> escritor.escreve_arquivo(dsv)
escreve_arquivo(dsvagua: inewave.newave.modelos.dsvagua.DSVAgua)[código fonte]¶

Realiza a escrita de um arquivo dsvagua.dat.
