Cargas Adicionais (c_adic.dat)

As configurações de cargas adicionais do NEWAVE , localizadas no arquivo geralmente denominado c_adic.dat, são armazenadas na classe:

class CAdic(data=Ellipsis)[código fonte]

Armazena os dados de entrada do NEWAVE referentes às cargas adicionais.

property cargas: Optional[DataFrame]

Tabela com as cargas adicionais por mês/ano e por subsistema para cada razão de carga adicional. As colunas são:

  • Código Subsistema (int)

  • Nome Subsistema (str)

  • Razão (str)

  • Ano (str)

  • Janeiro (float)

  • Fevereiro (float)

  • …

  • Dezembro (float)

Retorna

A tabela como um DataFrame

Tipo de retorno

pd.DataFrame | None
