Função de Produção Vento-Potência de Usina Eólicas (eolica-fte.csv)

As informações das funções de produção das usinas eólicas no NEWAVE, localizadas no arquivo geralmente denominado eolica-fte.csv, são armazenadas na classe:

class EolicaFTE(data=Ellipsis)[código fonte]

Armazena os dados de entrada do NEWAVE referentes às funções de transferência vento-potência.

append_registro(registro: Register)[código fonte]

Adiciona um registro ao arquivo na última posição. Este método existe para retrocompatibilidade e deve ser substituído quando for suportado na classe RegisterFile.

cria_registro(anterior: Register, registro: Register)[código fonte]

Adiciona um registro ao arquivo após um outro registro previamente existente.

Este método existe para retrocompatibilidade e deve ser substituído quando for suportado na classe RegisterFile.

deleta_registro(registro: Register)[código fonte]

Remove um registro existente no arquivo.

Este método existe para retrocompatibilidade e deve ser substituído quando for suportado na classe RegisterFile.

eolica_funcao_producao(codigo_eolica: Optional[int] = None, data_inicial: Optional[datetime] = None, data_final: Optional[datetime] = None, coeficiente_linear: Optional[float] = None, coeficiente_angular: Optional[float] = None) → Optional[Union[RegistroEolicaFTE, List[RegistroEolicaFTE]]][código fonte]

Obtém um registro que contém a função de produção vento-geração para um período de tempo.

Parâmetros

  • codigo_eolica (int | None) – código que especifica a usina

  • codigo_submercado (int | None) – código que especifica o submercado

Retorna

Um ou mais registros, se existirem.

Tipo de retorno

RegistroEolicaFTE | list[RegistroEolicaFTE] | None

preppend_registro(registro: Register)[código fonte]

Adiciona um registro ao arquivo na primeira posição. Este método existe para retrocompatibilidade e deve ser substituído quando for suportado na classe RegisterFile.

As informações existentes em cada um dos campos são armazenadas em modelos próprios para cada um.

class RegistroEolicaFTE(previous=None, next=None, data=None)[código fonte]

Registro que contém as informações da função de produção linear vento-potência para as usinas eólicas.
