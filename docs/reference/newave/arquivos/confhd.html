Configurações das Hidrelétricas (confhd.dat)¶

Visão geral do modelo¶

As configurações das hidrelétricas do NEWAVE, localizadas no arquivo geralmente denominado confhd.dat, são armazenadas na classe:

class Confhd(usinas: Dict[int, inewave.newave.modelos.confhd.UHEConfhd])[código fonte]¶

Armazena os dados de entrada do NEWAVE referentes às configurações das usinas hidrelétricas.

Esta classe lida com informações de entrada fornecidas ao NEWAVE e que podem ser modificadas através do arquivo modif.dat.

Parâmetros

  • usinas: Dict[int, UHEConfhd]

Leitura¶

A leitura do arquivo confhd.dat é feita através da classe LeituraConfhd.

class LeituraConfhd(diretorio: str)[código fonte]¶

Realiza a leitura do arquivo confhd.dat existente em um diretório de entradas do NEWAVE.

Esta classe contém o conjunto de utilidades para ler e interpretar os campos de um arquivo confhd.dat, construindo um objeto Confhd cujas informações são as mesmas do confhd.dat.

Este objeto existe para retirar do modelo de dados a complexidade de iterar pelas linhas do arquivo, recortar colunas, converter tipos de dados, dentre outras tarefas necessárias para a leitura.

Uma vez realizada a leitura do arquivo, as informações são guardadas internamente no atributo confhd.

Exemplos

>>> diretorio = "~/documentos/.../deck"
>>> leitor = LeituraConfhd(diretorio)
>>> leitor.le_arquivo()
# Ops, esqueci de pegar o objeto
>>> confhd = leitor.confhd
le_arquivo(nome_arquivo='confhd.dat') → inewave.newave.modelos.confhd.Confhd[código fonte]¶

Faz a leitura do arquivo confhd.dat.

Escrita¶

A escrita do arquivo confhd.dat é feita através da classe EscritaConfhd.

class EscritaConfhd(diretorio: str)[código fonte]¶

Realiza a escrita do arquivo confhd.dat em um diretório de entradas do NEWAVE.

Esta classe contém o conjunto de utilidades para escrever os campos de um arquivo confhd.dat, a partir de um objeto Confhd.

Este objeto existe para retirar do modelo de dados a complexidade de armazenar as strings auxiliares do arquivo, desenhar tabelas, dentre outras tarefas associadas à escrita.

Se o diretório de escrita não existir, ele será criado.

Exemplos

>>> diretorio = "~/documentos/.../deck"
>>> # confhd é do tipo Confhd
>>> escritor = EscritaConfhd(diretorio)
>>> escritor.escreve_arquivo(confhd)
escreve_arquivo(confhd: inewave.newave.modelos.confhd.Confhd)[código fonte]¶

Realiza a escrita de um arquivo confhd.dat.
