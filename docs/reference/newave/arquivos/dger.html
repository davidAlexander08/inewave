Dados Gerais (dger.dat)

Os dados gerais de execução do NEWAVE, localizados no arquivo geralmente denominado dger.dat, são armazenados na classe:

class DGer(data=Ellipsis)[código fonte]

Classe para armazenar dados gerais de uma execução do NEWAVE.
