Histórico de Vento das Usinas Eólicas (hist-ventos.csv)

Visão geral do modelo

As informações dos históricos de vento das usinas eólicas com os submercados no NEWAVE, localizadas no arquivo geralmente denominado hist-ventos.csv, são armazenadas na classe:

class EolicaHistorico(data=Ellipsis)[código fonte]

Armazena os dados de entrada do NEWAVE referentes ao histórico de ventos das usinas eólicas.

append_registro(registro: Register)[código fonte]

Adiciona um registro ao arquivo na última posição. Este método existe para retrocompatibilidade e deve ser substituído quando for suportado na classe RegisterFile.

cria_registro(anterior: Register, registro: Register)[código fonte]

Adiciona um registro ao arquivo após um outro registro previamente existente.

Este método existe para retrocompatibilidade e deve ser substituído quando for suportado na classe RegisterFile.

deleta_registro(registro: Register)[código fonte]

Remove um registro existente no arquivo.

Este método existe para retrocompatibilidade e deve ser substituído quando for suportado na classe RegisterFile.

eolica_historico_vento(codigo_eolica: Optional[int] = None, data_inicial: Optional[datetime] = None, data_final: Optional[datetime] = None, velocidade: Optional[float] = None, direcao: Optional[float] = None) → Optional[Union[RegistroEolicaHistoricoVento, List[RegistroEolicaHistoricoVento]]][código fonte]

Obtém um registro que contém a o valor de vento para um intervalo do histórico.

Parâmetros

  • codigo_eolica (int | None) – código da usina eólica

  • data_inicial (datetime | None) – data de início do registro histórico

  • data_final (datetime | None) – data de fim do registro histórico

  • velocidade (float | None) – velocidade do vento

  • direcao (float | None) – direção do vento

Retorna

Um ou mais registros, se existirem.

Tipo de retorno

RegistroEolicaHistoricoVento | list[RegistroEolicaHistoricoVento] | None

eolica_historico_vento_horizonte(data_inicial: Optional[datetime] = None, data_final: Optional[datetime] = None) → Optional[Union[RegistroEolicaHistoricoVentoHorizonte, List[RegistroEolicaHistoricoVentoHorizonte]]][código fonte]

Obtém um registro que contém o horizonte de um histórico.

Parâmetros

  • data_inicial (datetime | None) – data de início do histórico

  • data_final (datetime | None) – data de fim do histórico

Retorna

Um ou mais registros, se existirem.

Tipo de retorno

RegistroEolicaHistoricoVentoHorizonte | list[RegistroEolicaHistoricoVentoHorizonte] | None

preppend_registro(registro: Register)[código fonte]

Adiciona um registro ao arquivo na primeira posição. Este método existe para retrocompatibilidade e deve ser substituído quando for suportado na classe RegisterFile.

Campos

As informações existentes em cada um dos campos são armazenadas em modelos próprios para cada um.

class RegistroEolicaHistoricoVentoHorizonte(previous=None, next=None, data=None)[código fonte]

Registro que contém o horizonte de históricos de vento.

class RegistroEolicaHistoricoVento(previous=None, next=None, data=None)[código fonte]

Registro que contém os históricos de vento para o horizonte considerado.
