Energia Armazenada Final Percentual para o SIN (earmfpsin.out)

Visão geral do modelo

Os valores de energias armazenadas finais percentuais para o SIN, localizados no arquivo earmfpsin.out, são armazenados na classe:

class EarmfpSIN(data=Ellipsis)[código fonte]

Armazena os dados das saídas referentes às energias armazenadas finais para o SIN e em % da energia armazenável máxima.

Esta classe lida com as informações de saída fornecidas pelo NWLISTOP e reproduzidas nos earmfpsin.out, onde x varia conforme o submercado em questão.

property energias: Optional[DataFrame]

Tabela com as energias armazenadas percentuais por série e por mês/ano de estudo.

Retorna

A tabela das energias armazenadas.

Tipo de retorno

Optional[pd.DataFrame]
