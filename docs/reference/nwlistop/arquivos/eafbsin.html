Energia Afluente Bruta para o SIN (eafbsin.out)

Os valores de energias afluentes brutas para o SIN, localizados nos arquivos geralmente denominados eafbsin.out, são armazenados na classe:

class EafbSIN(data=Ellipsis)[código fonte]

Armazena os dados das saídas referentes à energia natural afluente para o SIN.

Esta classe lida com as informações de saída fornecidas pelo NWLISTOP e reproduzidas nos eafbsin.out.
