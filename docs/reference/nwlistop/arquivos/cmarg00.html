Custos Marginais (cmarg00x.out)¶

Visão geral do modelo¶

Os valores de custos marginais por patamar, localizados nos arquivos geralmente denominados cmarg00x.out, onde x varia confomrme o submercado, são armazenados na classe:

class Cmarg00(mes_pmo: int, ano_pmo: int, versao_newave: str, submercado: str, custos_patamares: Dict[int, numpy.ndarray])[código fonte]¶

Armazena os dados das saídas referentes aos custos marginais de operação por patamar, por submercado.

Esta classe lida com as informações de saída fornecidas pelo NWLISTOP e reproduzidas nos cmarg00x.out, onde x varia conforme o submercado em questão.

Parâmetros

  • mes_pmo: int

  • ano_pmo: int

  • versao_newave: str

  • submercado: str

  • custos_patamares: Dict[int, np.ndarray]

custos_medios_por_ano(patamar: inewave.newave.modelos.patamar.Patamar) → Dict[int, numpy.ndarray][código fonte]¶

Custos médios para cada ano de estudo.

Parâmetros

Patamar

Retorna

Dict[int, np.ndarray]

Sobre

Recebe um objeto Patamar para realizar a ponderação de cada cenários pelos devidos valores. O acesso é feito com [ano] e a saída é uma array 2-D do numpy com os valores médios de cada cenário e mês.

Leitura¶

A leitura dos arquivos cmarg00x.out é feita através da classe LeituraCmarg00.

class LeituraCmarg00(diretorio: str)[código fonte]¶

Realiza a leitura dos arquivos cmarg00x.out existentes em um diretório de saídas do NEWAVE.

Esta classe contém o conjunto de utilidades para ler e interpretar os campos de arquivos cmarg00x.out, construindo objetos Cmarg00 cujas informações são as mesmas dos arquivos.

Este objeto existe para retirar do modelo de dados a complexidade de iterar pelas linhas do arquivo, recortar colunas, converter tipos de dados, dentre outras tarefas necessárias para a leitura.

Uma vez realizada a leitura do arquivo, as informações são guardadas internamente no atributo cmargs.

Exemplos

>>> diretorio = "~/documentos/.../deck"
>>> leitor = LeituraCmarg00(diretorio)
>>> leitor.le_arquivos()
# Ops, esqueci de pegar o objeto
>>> cmarg_sudeste = leitor.cmargs['SUDESTE']
le_arquivos() → Dict[str, inewave.nwlistop.modelos.cmarg00.Cmarg00][código fonte]¶

Lê os arquivos cmarg00x.out em um diretório.
