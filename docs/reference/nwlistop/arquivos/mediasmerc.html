Médias por Submercado (MEDIAS-MERC.CSV)¶

Visão geral do modelo¶

Os valores de médias por submercado, localizados no arquivo geralmente denominado MEDIAS-MERC.CSV, são armazenados na classe:

class MediasMerc(mes_pmo: int, tabela: numpy.ndarray)[código fonte]¶

Classe responsável por lidar com o armazenamento dos dados das saídas do NWListOP referentes às médias de diversas variáveis agrupadas por submercado, existentes no arquivo MEDIAS-MERC.CSV.

property custo_deficit¶

Retorna os custos de déficit (R$) para cada um dos submercados. A série histórica retornada é dividida mensalmente, para todo o período de estudo.

property custo_marginal_operacao¶

Retorna os custos marginais de operação (R$) para cada um dos submercados. A série histórica retornada é dividida mensalmente, para todo o período de estudo.

property custo_termica¶

Retorna os custos de térmicas (R$) para cada um dos submercados. A série histórica retornada é dividida mensalmente, para todo o período de estudo.

property deficit¶

Retorna os déficits de carga em valores absolutos (MWmed) para cada um dos submercados. A série histórica retornada é dividida mensalmente, para todo o período de estudo.

property energia_controlavel_corrigida¶

Retorna as energias controláveis corrigidas em valores absolutos (MWmed) para cada um dos submercados. A série histórica retornada é dividida mensalmente, para todo o período de estudo.
