Médias por Submercado (MEDIAS-MERC.CSV)¶

Visão geral do modelo¶

Os valores de médias por submercado, localizados no arquivo geralmente denominado MEDIAS-MERC.CSV, são armazenados na classe:

class MediasMerc(mes_pmo: int, tabela: numpy.ndarray)[código fonte]¶

Armazena os dados das saídas referentes às médias de diversas variáveis agrupadas por submercado.

Esta classe lida com as informações de saída fornecidas pelo NWLISTOP e reproduzidas nos MEDIAS-MERC.CSV.

  • tabela: np.ndarray

Leitura¶

A leitura do arquivo MEDIAS-MERC.CSV é feita através da classe LeituraMediasMerc.

class LeituraMediasMerc(diretorio: str)[código fonte]¶

Realiza a leitura do arquivo MEDIAS-MERC.CSV existente em um diretório de saídas do NEWAVE.

Esta classe contém o conjunto de utilidades para ler e interpretar os campos do arquivo MEDIAS-MERC.CSV, construindo um objeto MediasMerc cujas informações são as mesmas do arquivo.

Este objeto existe para retirar do modelo de dados a complexidade de iterar pelas linhas do arquivo, recortar colunas, converter tipos de dados, dentre outras tarefas necessárias para a leitura.

Uma vez realizada a leitura do arquivo, as informações são guardadas internamente no atributo medias.

Exemplos

>>> diretorio = "~/documentos/.../deck"
>>> leitor = LeituraMediasMerc(diretorio)
>>> leitor.le_arquivo()
# Ops, esqueci de pegar o objeto
>>> medias = leitor.medias
